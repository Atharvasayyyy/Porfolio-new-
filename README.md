# Portfolio New

Live site: https://atharvasayyyy.github.io/Porfolio-new-

## Overview

This is a personal portfolio website built with React and Vite. It presents a full-stack and Gen-AI focused profile, selected projects, technical stack, experience, and contact links.

## Features

- Responsive single-page layout
- Hero section with profile image and CV download
- Projects, experience, education, and contact sections
- GitHub Pages-friendly asset paths and build setup

## Tech Stack

- React
- Vite
- Lucide React
- HTML and CSS

## Getting Started

### Prerequisites

- Node.js 18 or newer
- npm

### Install

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Deployment

The project is configured for GitHub Pages using a Vite base path and a GitHub Actions workflow in `.github/workflows/deploy.yml`.

## Folder Structure

```text
index.html
package.json
public/
	assets/
src/
	main.jsx
	styles.css
vite.config.js
.github/workflows/deploy.yml
```
